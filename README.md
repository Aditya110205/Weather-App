Weather App

A simple and responsive Weather Application built using HTML, and JavaScript.
It fetches real-time weather data using a public weather API and displays temperature, humidity, wind speed, and weather conditions for any city.

Features
Search weather by city name
Real-time temperature, humidity, and wind details
Dynamic weather icons
Responsive UI
Fast and lightweight

Tech Stack
HTML5
JavaScript (Vanilla JS)
OpenWeatherMap API

Project Structure
Weather-App/
│── index.html
│── script.js
│── README.md


How It Works
User enters a city name.
JavaScript sends a request to the OpenWeather API.

API returns JSON weather data.

UI updates dynamically with temperature, humidity, wind speed, etc.
